About This Project

Learn more about the data, technology, and how recommendations work.

Data Source

This movie recommender is built using the TMDb 5000 dataset, which contains information about 5,000 movies from The Movie Database (TMDb). The dataset includes details such as movie titles, overviews, genres, cast, directors, and ratings.

Movies are ranked by a calculated 'quality_score' based on vote average and vote count to highlight high-quality films.

How Recommendations Work

The recommender uses content-based filtering. When you search for a movie, it analyzes the genres, cast, and director of that movie and finds similar movies based on shared features.

Cosine similarity is computed on TF-IDF vectors of movie overviews combined with genre, cast, and director information to generate recommendations.

Recommendations are explained by highlighting common directors, cast members, or genres between the source movie and the recommended ones.

Technologies Used

  • Flask: A lightweight web framework for Python used to build the web application.
  • Pandas: A powerful data manipulation library for handling and analyzing the movie dataset.
  • Scikit-learn: Used for computing TF-IDF and cosine similarity for recommendations.
  • Tailwind CSS: A utility-first CSS framework for styling the user interface.
  • Jinja2: Templating engine used by Flask for rendering dynamic HTML pages.

Features

  • Search for movies by title and get personalized recommendations.
  • Browse movies by genre, with top lists and full catalogs.
  • Surprise Me! feature for discovering random high-quality movies.
  • Browse movies by director or actor.
  • Paginated catalogs for easy navigation.
