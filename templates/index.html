About This Project

This project solves the "cold-start" problem in movie recommendation. By analyzing metadata (Directors, Cast, Genres), we use a Weighted Content-Based approach to find semantic similarities without needing user history.

View Code Data Source: TMDB 5000 (Kaggle)

ðŸ§ª Why Trust Our Ratings?

Most sites just use a raw average (1-10). But a movie with one 10/10 vote isn't better than a movie with 10,000 8/10 votes. We use a Bayesian Average (the same math IMDb uses) to weigh ratings against vote counts, ensuring our 'Top Rated' lists actually feature high-quality films.

Tech Stack

  • Flask (Web Framework)
  • Pandas & Scikit-Learn (Data & ML)
  • TF-IDF & Cosine Similarity (Algorithms)
  • Tailwind CSS (Styling)

Features

  • Bayesian Quality Scoring
  • Weighted "Soup" Vectorization
  • Catalog Pagination
  • Director/Actor Browsing
